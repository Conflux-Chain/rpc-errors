## InvalidParamsError

InvalidParamsError occurs when incorrect parameters are provided. To correct this error, follow the instructions in the error message to adjust the parameters accordingly.

1. **Invalid parameter type**

>  "Invalid params: invalid type: integer `1`, expected an epoch number or 'latest_mined', 'latest_state', 'latest_checkpoint', 'latest_finalized', 'latest_confirmed' or 'earliest'."

2. **EpochNumberTooLargeError**

> You need to pass a `hex` string or epoch strings, not a `number`.

3. **EmptyEpochStringError**

> "Invalid params: Invalid epoch number: cannot parse integer from empty string."

4. **InvalidDigitEpochError**

> "Invalid params: Invalid epoch number: invalid digit found in string."

5. **MissingHexPrefixError**

> "Invalid params: Invalid epoch number: missing 0x prefix."

6. **InvalidHashTypeError**

> "... expected a (both 0x-prefixed or not) hex string with length of 64."

7. **NonExistentBlockHeaderError**

> "Specified block header does not exist"

8. **InvalidBase32AddressError**

> "nvalid params: Invalid base32 address: input 0x error zero or multiple prefixes."

9. **InvalidSize160AddressError**

> "Invalid base32 address: input ... not a SIZE_160 address.""

10. **SpecifiedEpochNotExecutedError**

> "Specified epoch 9007199254740991 is not executed, the latest state epoch is 0"
