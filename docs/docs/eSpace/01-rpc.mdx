## web3_clientVersion

Returns the current version of the chain client. 

### Errors

This method does not return any errors.



## net_version

Returns the current network id.

### Errors

This method does not return any errors.



## eth_protocolVersion

Returns the current ethereum protocol version.

### Errors

This method does not return any errors.


## eth_chainId

Returns the currently configured chain ID.

### Errors

This method does not return any errors.


## eth_gasPrice

Returns the current price per gas in wei.

### Errors

This method does not return any errors.


## eth_blockNumber

Returns the number of the most recent block.

### Errors

This method does not return any errors.


## eth_getBalance

Returns the balance of the account of a given address.

### Errors


1.  [InvalidParamsError](./errors#invalidparamserror)

1.  [InvalidParamsError](./errors#invalidparamserror)
    *   **InvalidParamsError**
    *   **MissingHexPrefixError**
    *   **InvalidBlockTypeError**
    *   **SpecifiedBlockNotExecutedError**
    *   **EmptyBlockStringError**
    *   **InvalidDigitError**

## eth_getStorageAt

Returns the value from a storage position at a given address, or in other words, returns the state of the contract's storage, which may not be exposed via the contract's methods.

### Errors

1.  [InvalidParamsError](./errors#invalidparamserror)
    *   **InvalidParamsError**
    *   **MissingHexPrefixError**
    *   **InvalidBlockTypeError**
    *   **SpecifiedBlockNotExecutedError**
    *   **EmptyBlockStringError**
    *   **InvalidDigitError**
